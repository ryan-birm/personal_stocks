# Backend API

Python FastAPI server

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Environment Variables

```env
SUPABASE_URL=your_url
SUPABASE_ANON_KEY=your_key
ALLOWED_ORIGINS=http://localhost:5173
PORT=8000
```

## API Endpoints

- `GET /` - Health check
- `GET /health` - Health check
